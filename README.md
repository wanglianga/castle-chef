# 🏰 古堡厨师 - Castle Chef

一款基于 Phaser 3 的快节奏动作订单游戏。玩家扮演古堡厨房的厨师，在机关移动的厨房里完成宴会订单，切菜、烤肉、调酱、装盘、送餐，同时躲避走廊陷阱。

## 原始需求

> 请用 Phaser 制作古堡厨师订单动作游戏，玩家在一座机关会移动的古堡厨房里完成宴会订单，要切菜、烤肉、调酱、装盘、送餐并避开走廊陷阱。厨房台面、炉火、食材箱、传送窗和贵宾桌会改变位置，玩家需要在倒计时内组合菜品，同时避免烧焦、送错桌、漏配酱汁或被机关挡住路线。手感要爽快，订单提示、食材状态、受伤硬直、连击奖励和失误反馈都要明显，短局结束也能给玩家一个想再冲高分的理由。

## 游戏特色

- **爽快操作**：WASD/方向键移动，Shift 冲刺，空格交互
- **食材加工**：切菜、烤肉、调酱、装盘，多步骤组合菜品
- **机关陷阱**：移动台面、尖刺陷阱、旋转障碍、传送窗
- **连击奖励**：连续完成订单获得分数加成，最高 2 倍
- **视觉反馈**：受伤闪烁、订单紧迫抖动、飘字提示、连击火焰效果
- **短局重玩**：每局 3 分钟，星级评分，高分记录激励再挑战

## 启动方式

### 前置要求

- Node.js >= 16
- npm 或 pnpm
- Docker & Docker Compose（可选，用于容器化部署）

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务

```bash
npm run dev
```

访问地址：http://localhost:5173

### 生产构建

```bash
npm run build
npm run preview
```

预览访问地址：http://localhost:4173

## Docker 一键启动

### 前置要求

- Docker Engine 20.10+
- Docker Compose v2+

### 启动命令

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:8080

停止服务：

```bash
docker compose down
```

## 游戏机制

### 食材系统

| 食材 | 加工方式 | 成品 |
|------|---------|------|
| 🥩 生肉 | 🔥 烤炉 3 秒 | 🍖 烤肉 |
| 🥕 蔬菜 | 🔪 切菜台 2 秒 | 🥗 切好的菜 |
| 🍞 面包 | 🔪 切菜台 1 秒 | 🍞 切片面包 |
| 🍅 番茄 | 🔪 切 + 🥣 调酱 | 🍅 番茄酱 |
| 🧀 奶酪 | 🔪 切菜台 1 秒 | 🧀 奶酪片 |

### 订单类型

- 经典烤肉：烤肉 + 切好的菜（100 分）
- 田园沙拉：切好的菜 + 番茄酱（80 分）
- 芝士三明治：切片面包 + 奶酪片 + 番茄酱（150 分）
- 烤肉大拼盘：烤肉 x2 + 切片面包（180 分）
- 豪华盛宴：烤肉 + 切好的菜 + 奶酪片（200 分）

### 计分规则

- 每连击 +10% 分数加成，最高 2 倍
- 送错桌：-50 分，连击清零
- 食材烧焦：-30 分，连击清零
- 陷阱受伤：-20 分，连击清零，硬直 1 秒，菜品掉落

### 宾客优先级系统

| 宾客 | 优先级 | 耐心 | 分数倍率 | 小费倍率 | 连锁效果 |
|------|--------|------|----------|----------|----------|
| 👑 国王 | ★★★★ 高 | 0.7x 短 | x2.0 | x3.0 | 皇家盛宴：小费+50%，厨房速度+30% |
| ⚔️ 骑士 | ★★★ 中高 | 0.85x 中短 | x1.5 | x1.5 | 战士之魂：连击额外+20% |
| ⚗️ 炼金术士 | ★★★ 中高 | 1.0x 中 | x1.8 | x2.5 | 神秘嬗变：随机20%-60%全属性加成 |
| 🎵 乐师 | ★★ 中 | 1.3x 长 | x1.2 | x2.0 | 旋律之流：订单刷新+40% |
| 👶 小孩 | ★ 低 | 0.5x 极短 | x0.8 | x0.5 | 顽童胡闹：订单打乱，厨房速度-20% |

### 连锁效果系统

完成特定宾客订单会触发持续一定时间的全局增益效果：

- **皇家盛宴**（30秒）：小费+50%，厨房速度+30%
- **战士之魂**（20秒）：连击倍率额外+20%
- **旋律之流**（25秒）：订单刷新速度+40%
- **神秘嬗变**（15秒）：随机获得20%-60%的全属性加成
- **顽童胡闹**（10秒）：订单顺序打乱，厨房速度-20%

### 新机关陷阱

| 机关 | 效果 | 应对策略 |
|------|------|----------|
| 🔄 旋转地板 | 周期性旋转并推动玩家，中心区域造成伤害 | 观察节奏，快速穿过 |
| 🔥 炉火波动 | 火焰强度周期性变化，影响烹饪速度，高火时靠近会受伤 | 低火时快速通过，高火时远离 |
| 🔄 传送窗换位 | 站在换位机关旁按空格可交换两个餐桌的订单 | 订单超时前紧急调整 |
| 🔪 滑动刀架 | 刀具沿轨道快速滑动 | 观察移动规律，找准时机穿过 |

### 机关躲避奖励

- 成功躲过机关可获得 **+30% 移动速度** 奖励，持续 3 秒
- 速度奖励可叠加，最高 **3 层（+90%）**
- 撞到陷阱会清除所有速度加成，并掉落手上的菜品
- 结算时统计总躲避次数，作为评价依据
